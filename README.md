## Commencez à contribuer

### 1. Récupérez le contenu du site avec git

```bash
git clone https://github.com/Alternumerica/docs.alternumerica.org.git
```

### 2. Installez les dépendences

```bash
npm install
```

### 3. Commencez un serveur de développement

```bash
npm run start
```
